Linux简单使用

2019-09-24 Linux

基本命令

切换目标 
cd

查看当前工作目录

pwd

查看当前目录下的内容

ls

查询磁盘的使用情况

df -h

查看当前目录的使用情况

du -sh

切换目录

cd	家目录
  cd /	根目录
  cd - 上次工作目录
  cd ..

创建目录

mkdir
mkdir -p 递归创建

创建文件

tocuh

使用yum安装jdk

yum search java-1.8
yum install java-1.8.0-openjdk
yum install java-1.8.0-openjdk-devel

安装tomcat

下载 
	wget http://mirrors.tuna.tsinghua.edu.cn/apache/tomcat/tomcat-8/v8.5.43/bin/apache-tomcat-8.5.43.tar.gz

解压缩

	tar -zxvf apache-tomcat-8.5.43.tar.gz

启动

	执行bin目录下的startup.sh文件
	./startup.sh

查看日志

	tail -f logs/catalina.out

停止

	执行bin目录下的shutdown.sh

hell连接CentOS

mySQL

安装： 
	yum install mariadb-server

启动：

	systemctl start mariadb

设置root密码：

	mysqladmin -u root password '123456'

使用root登录：

	mysql -uroot -p123456

部署SpringBoot的应用程序

将项目打包

	mvn package

导出数据

	使用Navicat导出SQL文件

导入数据

	使用Navicat导入SQL文件

将打包好的jar文件上传到服务器

	使用xftp上传文件

在服务器启动jar文件

	java -jar xxx.jar

后台启动：

	nohup java -jar xxx.jar &

通过进程结束程序

查看进程id： 
	ps -ef | grep java
	kill 进程id

允许mysql的root用户远程登录

GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY '123456';
GRANT ALL PRIVILEGES ON *.* TO 'root'@'localhost' IDENTIFIED BY '123456';
FLUSH PRIVILEGES;

部署war

  1. 将项目使用Maven打包为war
  2. 将war文件上传到服务器的tomcat下webapps目录
  3. 启动tomcat

CentOS7中配置服务的开启自启

  1. 检查服务当前状态

     	systemctl status firewalld.service

  2. 启动服务

     	systemctl start firewalld.service

  3. 停止服务

     	systemctl stop firewalld.service

  4. 禁止服务开机自启

     	systemctl disable firewalld.service

  5. 设置服务开机自启

     	systemctl enable firewalld.service

    设置MariaDB开机自启

     	systemctl enable mariadb

在CentOS7中安装Docker

  1. 系统版本要求
    docker对系统的内核版本最低要求是3.10+

  2. 查看当前系统的内核版本

     uname -r

  3. 使用yum安装docker

     yum install docker

  4. docker的启动和停止
    启动：

     systemctl start docker

    停止：

     systemctl stop docker

  5. docker 检查版本

     docker -v

  6. 使用docker安装tomcat
    搜索

     docker search tomcat

    下载(拉取)镜像

     docker pull tomcat

    启动镜像

     docker run --name tomcat1 -p8080:8080 -d tomcat
 docker run --name tomcat2 -p8081:8080 -d tomcat
 docker run --name tomcat3 -p8082:8080 -d tomcat

  7. docker常用操作
    查看正则运行的容器

     docker ps -a(查询全部，包含已经停止的)

    停止正在运行的容器

     docker stop 容器id

    启动容器

     docker start 容器id

    重启容器

     docker restart 容器id

    删除容器

     docker rm 容器id

  8. 进入容器

     docker exec -it 容器id /bin/bash

    在webapps下新建demo目录

     mkdir demo

    在demo目录创建index.jsp

     touch index.jsp

    修改index.jsp的内容

     echo '<%=new java.util.Date()%>' >> index.jsp

    访问：http://ip:8080/demo

  9. 使用docker部署war文件

     docker cp xx.war 容器id:/usr/local/tomcat/webapps/

  10. 使用docker安装Redis
    搜索

    docker search redis

    下载

    docker pull redis
